# # JavaTest2

build.sh分别放入了EXAM1,EXAM2,EXAM3三个文件夹中

1. 首先  git clone https://github.com/caojiameng/JavaTest2.git
2. 然后 cd JavaTest2

## EXAM1
要查看第一题则
``` markdown
cd Exam1
sh build.sh

```

## Exam2
要查看第二题则
``` markdown
cd Exam2
sh build.sh

```
## Exam3
要查看第三题则
``` markdown
cd Exam3
sh build.sh

```
如要查看其他题，在`sh build.sh`后记得`cd ..`返回JavaTest2文件夹再使用上述操作
